Frequently Asked Questions

What languages are supported?

All the languages! No, really, subti's list of languages is only limited by the AI models you want it to prompt. In general, this means that larger languages will work better, although LLM:s are really impressively broad in this sense. I think you can even use subti to learn Klingon, if you want to.

As for sounds, the image is a bit more concrete. subti uses Whisper for speech recognition -- you can see the supported languages here. For text-to-voice playback, subti will use whatever language you have installed, and if it cannot find the language specified, it uses tts-edge. See this page for a language-to-locale list.

Is subti free?

Yes, with some limitations. The free version of subti requires ads and using it online with our API keys is limited to gpt-4o-mini and preset prompts. It also keeps tracks of your usage in tokens, and if your account is creating a deficit due to heavy usage, Firebase access can be blocked. You can still use the free version with your own API keys or locally installed AI models.

To get rid of the ads, you have to subscribe. Subscription is 3 $ per month. However, in the (highly unlikely) event that your subti firebase API usage reaches 2.5 $, it will block to the same limitations as the free version until you add 1 $ to that month's bill.

What about my data?

subti does not collect any of your data. In trial or subscription mode, requests are routed through firebase to the selected AI API. To hedge for abuse, subti keeps track of the number of tokens spent. In the unlikely case that you spend more tokens than the monthly cost of the subscription, you will be prompted to add 1 $ to the bill to cover the costs. In unlocked mode, subti is only keeping track of your login history.

What if I do not want to support American AI companies at this time?

subti supports using Mistral, which is based in France. If you are concerned about privacy, I suggest you install an offline model and set subti to use it. You can also connect with subti through a VPN.

Why do you have to have ads for the free version?

In addition to the costs of developing and maintaining the program, API requests cost money. Ads help cover these costs.
